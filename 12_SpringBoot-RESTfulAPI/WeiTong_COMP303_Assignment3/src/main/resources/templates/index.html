🍽️ Food Delivery Management System API Documentation v1.0
Quick Start
This API provides CRUD operations for Restaurant, MenuItem, and Order management. All endpoints return JSON format. Base URL: http://localhost:8080

🏪 Restaurant APIs

GET /restaurants

Get all restaurants

Method: GET URL: http://localhost:8080/restaurants // No request body required
GET /restaurants/{id}

Get restaurant by ID

Method: GET URL: http://localhost:8080/restaurants/1 // Path Variable: id = 1
POST /restaurants

Create a new restaurant (ID auto-generated)

Method: POST URL: http://localhost:8080/restaurants Headers: Content-Type: application/json Body (raw JSON): { "restName": "Haidilao Hotpot", "cuisineType": "Chinese Hotpot", "city": "Chengdu", "rating": 4.8, "deliveryTime": 45.5 }
PUT /restaurants/{id}

Update restaurant by ID

Method: PUT URL: http://localhost:8080/restaurants/1 Headers: Content-Type: application/json Body (raw JSON): { "restName": "Updated Restaurant", "cuisineType": "Sichuan Cuisine", "city": "Beijing", "rating": 4.5, "deliveryTime": 30.0 }
DELETE /restaurants/{id}

Delete restaurant by ID

Method: DELETE URL: http://localhost:8080/restaurants/1 // Path Variable: id = 1
📦 Restaurant Model
{ "restId": 1, // Long, Auto-generated (@Id) "restName": "Haidilao", // String, Required "cuisineType": "Hotpot", // String, Optional "city": "Chengdu", // String, Optional "rating": 4.8, // double, Optional (0.0 - 5.0) "deliveryTime": 45.5 // double, Optional (minutes) }

🍔 MenuItem APIs

GET /menuitems

Get all menu items

Method: GET URL: http://localhost:8080/menuitems // No request body required
GET /menuitems/{id}

Get menu item by ID

Method: GET URL: http://localhost:8080/menuitems/1 // Path Variable: id = 1
GET /menuitems/restaurant/{restId}

Get menu items by restaurant ID

Method: GET URL: http://localhost:8080/menuitems/restaurant/1 // Path Variable: restId = 1
POST /menuitems

Create a new menu item (ID auto-generated)

Method: POST URL: http://localhost:8080/menuitems Headers: Content-Type: application/json Body (raw JSON): { "itemName": "Spicy Beef Slice", "category": "Meat", "price": 45.00, "availability": true, "restId": 1 }
PUT /menuitems/{id}

Update menu item by ID

Method: PUT URL: http://localhost:8080/menuitems/1 Headers: Content-Type: application/json Body (raw JSON): { "itemName": "Premium Beef Slice", "category": "Premium Meat", "price": 58.00, "availability": true, "restId": 1 }
PUT /menuitems/{id}/availability

Update menu item availability status

Method: PUT URL: http://localhost:8080/menuitems/1/availability Headers: Content-Type: application/json Body (raw JSON): { "availability": false }
DELETE /menuitems/{id}

Delete menu item by ID

Method: DELETE URL: http://localhost:8080/menuitems/1 // Path Variable: id = 1
📦 MenuItem Model
{ "menuItemId": 1, // Long, Auto-generated (@Id) "itemName": "Beef Slice", // String, Required "category": "Meat", // String, Optional "price": 45.00, // double, Required "availability": true, // boolean, Default: true "restId": 1 // Long, Required (Foreign Key to Restaurant) }

📋 Order APIs

GET /orders

Get all orders

Method: GET URL: http://localhost:8080/orders // No request body required
GET /orders/{id}

Get order by ID

Method: GET URL: http://localhost:8080/orders/1 // Path Variable: id = 1
POST /orders

Create a new order (ID and time auto-generated)

Method: POST URL: http://localhost:8080/orders Headers: Content-Type: application/json Body (raw JSON): { "customerName": "John Doe", "menuItemId": 1, "quantity": 2, "orderStatus": "Pending" } // orderTime is auto-set to current time
PUT /orders/{id}

Update order details (quantity, menuItem, etc.)

Method: PUT URL: http://localhost:8080/orders/1 Headers: Content-Type: application/json Body (raw JSON): { "customerName": "John Doe", "menuItemId": 2, "quantity": 3, "orderStatus": "Preparing" }
DELETE /orders/{id}

Cancel/Delete order by ID

Method: DELETE URL: http://localhost:8080/orders/1 // Path Variable: id = 1
📦 Order Model
{ "orderId": 1, // Long, Auto-generated (@Id) "customerName": "John Doe", // String, Required "menuItemId": 1, // Long, Required (Foreign Key to MenuItem) "quantity": 2, // int, Required "orderStatus": "Pending", // String, Default: "Pending" // Options: Pending // Preparing // Delivered "orderTime": "2024-01-15T10:30:00" // LocalDateTime, Auto-generated (creation time) }
