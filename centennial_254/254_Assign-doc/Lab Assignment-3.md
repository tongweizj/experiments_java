## Exercise-1

Write a recursive method to produce the following pattern:
```
*
**
***
****
***
**
*
```
Test the method by asking the user to enter the number of asterisks of the maximum line (for example, the user should enter 4 in this case).

## Exercise-2

Write **a recursive method** to that returns the number of vowels in a string. Test the method by asking the user to enter a string. 
编写一个递归方法返回字符串中元音的数量。通过要求用户输入字符串来测试该方法。
**Hint**: create a special method for checking if a character is a vowel.
## Exercise-3

Implement a **recursive method** that takes _(path, filename)_ as arguments and returns all entries of the file system rooted at the given _path_ having the given _file name_. Test the method with a real path, filename from your file system. 

**Hint**: Review use of the _java.io.File_ class and the week 5 examples.
